HTML (HyperText Markup Language)

, é a linguagem de marcação padrão para criar e estruturar o conteúdo de páginas da web, não sendo uma linguagem de programação. Ela utiliza uma série de elementos, definidos por tags, para organizar informações como textos, imagens, tabelas e links, instruindo os navegadores sobre como exibir e estruturar o conteúdo de um site.

CSS(Cascading Style Sheets ou Folhas de Estilo em Cascata)

, é uma linguagem usada para descrever a apresentação e o estilo de documentos escritos em linguagens de marcação como o HTML. Enquanto o HTML fornece a estrutura do conteúdo, o CSS é responsável por como esse conteúdo é apresentado visualmente na tela, controlando cores, fontes, espaçamentos, alinhamentos e layouts. Ele permite que os desenvolvedores separem a aparência do conteúdo de sua função estrutural, tornando os códigos mais limpos, fáceis de manter e permitindo a criação de layouts responsivos que se adaptam a diferentes dispositivos.

JavaScript (JS)

, é uma linguagem de programação essencial para a web, responsável por adicionar interatividade e dinamismo a páginas e aplicativos. Ela funciona como a "camada de comportamento" do desenvolvimento web, complementando o HTML (estrutura) e o CSS (design), e é interpretada diretamente pelos navegadores. Com JS, é possível criar desde menus interativos e galerias de fotos até gráficos complexos e jogos, tanto no lado do cliente (navegador) quanto no lado do servidor (através de ambientes como o Node.js).
